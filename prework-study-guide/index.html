HTML

  • The head element contains information about the webpage.
  • The Meta element contains the information for the browser.
  • The body element represents the visible content shown to the user.
  • H1 is only used once in a page, the rest are used in order to benefit screen readers.
  • Image information sits inside the image element brackets.
  • Viewport helps the webpage to adapt and fit the screen of the device.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Colours can be added as hex codes or standard references such as blue or yellow.
  • by linking the CSS file in the head section of the HTML we can effectively keep the files separate.

JavaScript

  • Javascript is executed from top to bottom so control flow needs to be considered.
  • Control flow is the order in which a computer executes code in a script.
  • A varaiable is a named container that allows us to store date in our code.
  • Boolean = a true or false input.
  • String = written words.
  • Functions can be called in when needed and allow us to execute a chunk of code.
